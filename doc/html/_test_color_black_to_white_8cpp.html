Detailed Description

Test program of the ColorBlackToWhite function.

Author
Alexandre CASANOVA–FRANGER, Gauthier LARMARQUE, Paul SIMORRE, Lucas VIVAS
Date
March 2018

This test is used to show us that our color function taking a float as parameter and returning an RGB triplet representing a color between black and white is working well. To demonstrate that, we take three float with the minimal expected value, the maximal expected value and the medium expected value and then pass them as parameter to our color function. The test is satisfied if the string corresponding of the color results are the same as expected on the three assertions int the testColor function.
