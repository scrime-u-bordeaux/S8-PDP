Detailed Description

Parser object used to get information from config file.

Author
Jérémy LIXANDRE
Date
July 2017

Parses a config file with filenames and functions. It is used in main.cpp to fill a Channel settings structure with the parameters given to the program. It is also used to prevent the use of too many occurencies of each element (ex: 2 lines with COLOR). In this case, the last line is kept.
