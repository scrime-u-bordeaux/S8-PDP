Code containing the coloration from black to white function.

Code containing the coloration from green to red function.

Author
Jérémy LIXANDRE
Date
July 2017

This color function is used to compute, from a float representing the correlation between two instruments, the RGB triplet corresponding to that value. The returned value will tend to be an RGB triplet representing black if the correlation is minimal or an RGB triplet representing white if the correlation is maximal.
