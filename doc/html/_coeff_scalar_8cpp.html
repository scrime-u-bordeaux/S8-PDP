Detailed Description

Code containing the scalar correlation function.

Author
Jérémy LIXANDRE
Date
July 2017

This is a correlation function used to return a correlation value corresponding of the scalar product of the two buffers containing the signals of two differents instruments for one frame of data. The volume and the coloration matrix will be affected by this value.
