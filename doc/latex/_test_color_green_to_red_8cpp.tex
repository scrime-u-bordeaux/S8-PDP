This test is used to show us that our color function taking a float as parameter and returning an \mbox{\hyperlink{class_r_g_b}{R\+GB}} triplet representing a color between red and green is working well. To demonstrate that, we take three float with the minimal expected value, the maximal expected value and the medium expected value and then pass them as parameter to our color function. The test is satisfied if the string corresponding of the color results are the same as expected on the three assertions int the test\+Color function.
