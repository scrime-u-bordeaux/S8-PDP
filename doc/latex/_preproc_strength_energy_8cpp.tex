\subsection{Detailed Description}
Code containing the preprocessing based on strength energy function. 

\begin{DoxyAuthor}{Author}
Jérémy L\+I\+X\+A\+N\+D\+RE 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
July 2017
\end{DoxyDate}
This is a preprocessing function returning a \mbox{\hyperlink{class_matrix}{Matrix}} witch has in every line the strength energy envelope of each instrument of the original \mbox{\hyperlink{class_matrix}{Matrix}} of signals given in parameter. It is almost the same as the energy enevelope process except that this functions returns, for each instrument, the difference of the energy process between two consecutives frames, or zero if the difference is negative.
