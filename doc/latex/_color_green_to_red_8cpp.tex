This color function is used to compute, from a float representing the correlation between two instruments, the \mbox{\hyperlink{class_r_g_b}{R\+GB}} triplet corresponding to that value. The returned value will tend to be an \mbox{\hyperlink{class_r_g_b}{R\+GB}} triplet representing black if the correlation is minimal or an \mbox{\hyperlink{class_r_g_b}{R\+GB}} triplet representing white if the correlation is maximal.
